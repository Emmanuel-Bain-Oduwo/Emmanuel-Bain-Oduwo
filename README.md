## 🏥 Kemirix — What I'm Building

> **`Rules decide. MedGemma explains.`**

**[Kemirix](https://kemirix.com)** is a clinical medication safety platform that ensures the rational and safe use of medications — helping doctors, pharmacists, and nurses prescribe and administer drugs safely by analyzing prescriptions against the patient's full clinical context: medical conditions, vitals, allergies, organ dysfunction, labs, and genetics.

| Component | Details | Status |
|-----------|---------|--------|
| 🧠 MedGemma 27B Fine-tuning | GCP TPU v5litepod-16 · SPMD/FSDPv2 · LoRA/PEFT | 🟡 Active |
| ⚙️ Backend API | FastAPI + asyncpg + Alembic + Redis + JWT · Railway | ✅ Deployed |
| 📊 DDI Dataset (Cat. 1) | 26 drugs · 334 pairs · 7 JSONL task types · 6 safety classes | ✅ Published |
| 🌐 Marketing Site | kemirix.com · Vercel frontend · Cloudflare DNS | ✅ Live |
| 🏥 PPB Kenya SaMD | Class II MDSW · Kenya Pharmacy & Poisons Board | 🎯 Aug 2026 |
| 🚀 Hospital Launch | KEMRI → Aga Khan Nairobi → KNH → Moi/AMPATH | 🎯 Jan 2027 |

**Accuracy target: 99.6% on answered queries** via refusal gating (~85% answered at high confidence, ~15% refused with structured uncertainty). Tier 1 APIs — OpenFDA, ClinPGx/PharmGKB, OncoKB — hold absolute veto power over all safety decisions.

**Dataset:** [`Oduwo/kemirix_category1_ddi`](https://huggingface.co/datasets/Oduwo/kemirix_category1_ddi) — 334 DDI pairs across 6 safety classes: `ABSOLUTE_CONTRAINDICATION` → `INSUFFICIENT_EVIDENCE`
